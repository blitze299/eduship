About Eduship

This is what we do.

Das Ziel von Eduship ist es digitale Bildungsveranstaltungen wie zb. Think Big, Code + Design etc. übersichtlich und katalogisiert auf einer Karte darzustellen. Außerdem soll es möglich sein sich direkt per Mail über zukünftige Events informieren zu lassen. Somit wollen wir erreichen, das noch mehr Jugendliche von solchen Veranstaltungen erfahren.
